# Kỹ năng Kể chuyện (Storytelling)

> Tạo ra những câu chuyện hấp dẫn khiến độc giả/khán giả không thể rời mắt

## Phạm vi áp dụng

- Sáng tác tiểu thuyết/truyện ngắn
- Kịch bản và storyboard truyện tranh
- Kịch bản phim/video ngắn
- Thiết kế cốt truyện game
- Storytelling cho thương hiệu

## Cấu trúc câu chuyện

### Cấu trúc ba hồi (Three-Act Structure)

```
┌─────────────────────────────────────────────────────────────────┐
│  Three-Act Structure - Cấu trúc ba hồi                          │
│                                                                 │
│  Hồi 1 (25%)         Hồi 2 (50%)          Hồi 3 (25%)          │
│  Setup              Confrontation        Resolution            │
│  ┌─────────┐       ┌─────────────┐       ┌─────────┐           │
│  │ Thiết lập│       │  Đối đầu    │       │ Giải quyết│         │
│  │ Thường   │       │  Leo thang  │       │ Cao trào  │         │
│  │ nhật     │       │  xung đột   │       │ Cân bằng  │         │
│  │ Sự kiện  │       │  Điểm giữa  │       │ mới       │         │
│  │ khơi mào │       │  chuyển     │       │           │         │
│  └────┬────┘       └──────┬──────┘       └────┬────┘           │
│       │                   │                   │                 │
│  ─────●───────────────────●───────────────────●─────→           │
│    Inciting          Midpoint            Climax                │
│    Event             Điểm chuyển         Cao trào               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Hành trình anh hùng (Hero's Journey)

```markdown
## 12 giai đoạn của Hành trình Anh hùng

### Khởi hành (Departure)

1. **Thế giới thường nhật** - Cuộc sống bình thường
2. **Lời kêu gọi phiêu lưu** - Sự kiện khơi mào
3. **Từ chối lời kêu gọi** - Do dự và sợ hãi
4. **Gặp người dẫn đường** - Nhận được chỉ dẫn
5. **Vượt qua ngưỡng cửa** - Bước vào thế giới mới

### Khai sáng (Initiation)

6. **Thử thách và đồng minh** - Kết bạn, đối mặt thách thức
7. **Tiếp cận hang động** - Đến gần khủng hoảng cốt lõi
8. **Thử thách khắc nghiệt** - Thử thách lớn nhất
9. **Phần thưởng** - Nhận được báu vật/năng lực

### Trở về (Return)

10. **Con đường hồi qui** - Trên đường trở về
11. **Tái sinh** - Thử thách cuối cùng
12. **Trở về với báu vật** - Cuộc sống mới sau khi thay đổi
```

### Xương sống câu chuyện (Story Spine)

```markdown
## Pixar Story Spine

Once upon a time there was **_. (Ngày xửa ngày xưa có...)
Every day, _**. (Mỗi ngày...)
One day **_. (Một ngày nọ...)
Because of that, _**. (Vì thế...)
Because of that, **_. (Vì thế...)
Because of that, _**. (Vì thế...)
Until finally **_. (Cho đến cuối cùng...)
And ever since then _**. (Và từ đó...)

Ví dụ:
Ngày xửa ngày xưa có một robot nhút nhát tên WALL-E.
Mỗi ngày nó dọn rác một mình.
Một ngày nọ, một robot tên EVE đến.
Vì thế, WALL-E phải lòng cô ấy và theo cô lên tàu vũ trụ.
Vì thế, nó vô tình kích hoạt kế hoạch trở về Trái Đất.
Vì thế, loài người cuối cùng trở về Trái Đất.
Cho đến cuối cùng, WALL-E và EVE ở bên nhau.
Và từ đó, Trái Đất bắt đầu phục hồi.
```

## Xây dựng nhân vật

### Các yếu tố cấu thành nhân vật

```
┌─────────────────────────────────────────────────────────────────┐
│  Mô hình tảng băng trôi của nhân vật                            │
│                                                                 │
│  ~~~~~~~~~~~~ Mặt nước ~~~~~~~~~~~~                             │
│                                                                 │
│  Tầng nhìn thấy (Biểu hiện bên ngoài)                          │
│  ├─ Ngoại hình, trang phục, cách nói                           │
│  ├─ Thói quen hành vi, kỹ năng                                 │
│  └─ Nghề nghiệp, địa vị xã hội                                 │
│                                                                 │
│  ~~~~~~~~~~~~ Dưới nước ~~~~~~~~~~~~                            │
│                                                                 │
│  Tầng ẩn giấu (Động lực nội tâm)                               │
│  ├─ Nỗi sợ và khát vọng                                        │
│  ├─ Chấn thương quá khứ                                        │
│  ├─ Niềm tin cốt lõi                                           │
│  └─ Khuyết điểm chết người (Fatal Flaw)                        │
│                                                                 │
│  Nhân vật tốt = Mục tiêu bên ngoài + Nhu cầu nội tâm + Trở ngại│
└─────────────────────────────────────────────────────────────────┘
```

### Bảng thiết kế nhân vật

```markdown
## Hồ sơ nhân vật

### Thông tin cơ bản

- Tên:
- Tuổi:
- Nghề nghiệp:
- Đặc điểm ngoại hình:

### Tầng nội tâm

- **Want (Mục tiêu bên ngoài)**: Muốn gì?
- **Need (Nhu cầu nội tâm)**: Thực sự cần gì?
- **Flaw (Khuyết điểm chết người)**: Điều gì cản trở?
- **Ghost (Chấn thương quá khứ)**: Chuyện gì đã xảy ra?

### Tính cách

- 3 đặc điểm tích cực:
- 3 đặc điểm tiêu cực:
- Câu nói miệng:
- Cử chỉ quen thuộc:

### Mạng lưới quan hệ

- Mối quan hệ với nhân vật chính:
- Nguồn gốc xung đột:
```

## Viết đối thoại

### Đặc điểm của đối thoại tốt

| Đặc điểm           | Giải thích                 | Ví dụ                                           |
| ------------------ | -------------------------- | ----------------------------------------------- |
| **Hàm ý**          | Ý nghĩa ngầm               | "Trời đẹp thật" (muốn kết thúc cuộc trò chuyện) |
| **Xung đột**       | Mục tiêu đối lập           | A muốn rời đi, B muốn giữ lại                   |
| **Cá nhân hóa**    | Phản ánh đặc điểm nhân vật | Cách nói của giáo sư vs trẻ em                  |
| **Đẩy cốt truyện** | Mỗi câu có mục đích        | Tiết lộ thông tin hoặc thay đổi quan hệ         |

### Kỹ thuật đối thoại

```markdown
## Nguyên tắc viết đối thoại

1. **Ít là nhiều**
   ❌ "Sáng nay tôi đi siêu thị, mua sữa và bánh mì."
   ✅ "Tủ lạnh hết đồ rồi."

2. **Hành động thay lời giải thích**
   ❌ "Tôi rất tức giận!" anh nói giận dữ.
   ✅ Anh đóng sầm cửa.

3. **Ngắt lời và im lặng**
   - Ngắt lời = Thế mạnh/cấp bách
   - Im lặng = Suy nghĩ/phản kháng

4. **Trả lời gián tiếp**
   "Anh có yêu em không?"
   ❌ "Có, anh yêu em."
   ✅ "Cà phê của em nguội rồi."
```

## Kịch bản truyện tranh

### Thiết kế storyboard

```
┌─────────────────────────────────────────────────────────────────┐
│  Format cơ bản storyboard truyện tranh                          │
│                                                                 │
│  Trang: P.XX                                                    │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ Ô 1 (lớn)                                                │   │
│  │ Loại cảnh: Toàn cảnh                                     │   │
│  │ Mô tả: Đường chân trời thành phố, hoàng hôn              │   │
│  │ Đối thoại: (Không có)                                    │   │
│  │ Hiệu ứng: Thiết lập không khí                            │   │
│  ├─────────────────────┬───────────────────────────────────┤   │
│  │ Ô 2                 │ Ô 3                               │   │
│  │ Loại: Cảnh trung    │ Loại: Cận cảnh                    │   │
│  │ Mô tả: Nhân vật     │ Mô tả: Đôi mắt nhân vật chính     │   │
│  │ chính đi trên phố   │                                   │   │
│  │ Đối thoại: ...      │ Đối thoại: "Cuối cùng cũng đến."  │   │
│  └─────────────────────┴───────────────────────────────────┘   │
│                                                                 │
│  Tham khảo loại cảnh:                                           │
│  Toàn cảnh → Thiết lập bối cảnh                                 │
│  Cảnh rộng → Hiển thị toàn thân nhân vật                        │
│  Cảnh trung → Cảnh đối thoại                                    │
│  Cận cảnh → Nhấn mạnh cảm xúc                                   │
│  Cận cảnh lớn → Chi tiết đồ vật hoặc biểu cảm                   │
└─────────────────────────────────────────────────────────────────┘
```

### Nhịp điệu truyện tranh

```markdown
## Kiểm soát nhịp điệu

### Tăng tốc (Căng thẳng/Hành động)

- Ô nhỏ và nhiều
- Bố cục đường chéo
- Đường tốc độ
- Ít đối thoại

### Giảm tốc (Cảm xúc/Suy nghĩ)

- Ô lớn hơn
- Khoảng trắng
- Cận cảnh biểu cảm
- Độc thoại nội tâm

### Bất ngờ khi lật trang

- Hình ảnh quan trọng đặt sau khi lật trang
- Tình tiết hồi hộp ở ô cuối trang phải
```

## Kịch bản video

### Format kịch bản

```markdown
## Mẫu kịch bản phim ngắn

Cảnh 1 - Nội cảnh/Quán café/Ngày

[Mô tả]
Ánh sắng mặt trời chiếu qua cửa kính lớn. Tiểu My (25 tuổi)
ngồi một mình ở góc, nhìn chằm chằm vào màn hình điện thoại.

[Hành động]
Tiểu My lướt điện thoại, biểu cảm từ kỳ vọng chuyển sang thất vọng.

[Đối thoại]
Tiểu My: (Tự nói) Lại đọc rồi không trả lời...

[Chuyển cảnh]
Cận cảnh màn hình điện thoại, hiển thị "Đang soạn tin..."

---

Cảnh 2 - Ngoại cảnh/Đường phố/Ngày

...
```

### Cấu trúc video

| Loại           | Độ dài     | Cấu trúc                        |
| -------------- | ---------- | ------------------------------- |
| **Video ngắn** | 15-60s     | Hook → Nội dung → CTA           |
| **YouTube**    | 8-15 phút  | Hook → Vấn đề → Giải pháp → CTA |
| **Phim ngắn**  | 5-15 phút  | Phiên bản rút gọn ba hồi        |
| **Micro film** | 15-40 phút | Ba hồi đầy đủ                   |

## Xây dựng thế giới quan

### Các yếu tố thế giới quan

```markdown
## Checklist xây dựng thế giới (Worldbuilding)

### Thiết lập cơ bản

- [ ] Bối cảnh thời đại (Quá khứ/Hiện đại/Tương lai)
- [ ] Môi trường địa lý
- [ ] Trình độ công nghệ
- [ ] Hệ thống ma thuật/siêu năng lực (nếu có)

### Cấu trúc xã hội

- [ ] Chế độ chính trị
- [ ] Hệ thống kinh tế
- [ ] Tầng lớp xã hội
- [ ] Phong tục văn hóa

### Quy tắc và giới hạn

- [ ] Điều gì có thể?
- [ ] Điều gì bị cấm?
- [ ] Hậu quả khi vi phạm quy tắc?

### Lịch sử

- [ ] Sự kiện lịch sử quan trọng
- [ ] Ảnh hưởng đến hiện tại như thế nào?
```

## Câu hỏi thường gặp

| Vấn đề              | Giải pháp                                     |
| ------------------- | --------------------------------------------- |
| Mở đầu quá chậm     | Bắt đầu từ hành động/xung đột                 |
| Phần giữa nhàm chán | Thêm bước ngoặt và chướng ngại                |
| Nhân vật phẳng      | Thêm khuyết điểm và mâu thuẫn                 |
| Kết thúc yếu        | Hồi đáp phần mở đầu, hoàn thiện cung nhân vật |
| Đối thoại cứng nhắc | Thêm hàm ý, giảm trần thuật trực tiếp         |
