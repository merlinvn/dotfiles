# Shorthand explanations (taken from cheat40)
# »      Forwards/rightwards
# «      Backwards/leftwards
# WORD   Space-delimited word
#
# Notation for key sequences:
#
# _      Leader (see `<leader>`)
# \      Localleader (see `<localleader>`)
# ‹r›    A register (", *, 0, 1, …)
#        See `registers`
# ‹Move› A motion or a text object
#        See `$VIMRUNTIME/doc/motion.txt`
#        See `text-objects`
# ‹P›    A regular expression (pattern)
#        See `$VIMRUNTIME/doc/pattern.txt`
# ‹Cmd›  An Ex command
#        See `$VIMRUNTIME/doc/cmdline.txt`
# ‹Var›  A variable or option name
#        See `variables`
#        See `$VIMRUNTIME/doc/options.txt`
# ‹n›    A (possibly optional) number
# ‹c›    A single character
# …      Any text
# «C-…»  Press ctrl and … together
# «Spc»  Press space bar
# «CR»   Press Enter
#
# Labels:
#
# N    Type in Normal mode
# C    Type in Command mode
# I    Type in Insert mode
# V    Type in Visual mode
# M    Motion (may be used by itself or
#      after an operator)
# T    Text object (use in Visual mode
#      or after an operator)
# O    Operator (use in Visual mode, or
#      before a motion or a text object)
# *    Acts on text under/after cursor
## mini.basic
Move down by visible lines with no [count]    | j     | Normal, Visual   
Move up by visible lines with no [count]      | k     | Normal, Visual   
Add [count] empty lines after cursor          | go    | Normal           
Add [count] empty lines before cursor         | gO    | Normal           
Copy to system clipboard                      | gy    | Normal, Visual   
Paste from system clipboard                   | gp    | Normal, Visual   
Visually select latest changed or yanked text | gV    | Normal           
Search inside current visual selection        | g/    | Visual           
Search forward for current visual selection   | *     | Visual           
Search backward for current visual selection  | #     | Visual           
Save and go to Normal mode                    | <C-s> | Normal, Visual, Insert       
Correct latest misspelled word                | <C-z> | Normal, Insert
